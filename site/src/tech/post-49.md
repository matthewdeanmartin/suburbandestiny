A good introductory article on [using Visual Source Safe](http://www.softsteel.co.uk/tutorials/vss/index.html) 6\.0\.

I think the Visual Studio team has made up for some of the worst VSS usability issues. I wouldn’t think of using VSS with anything less than VS 2005\.  Code repositories that use SQL as the back end would be better, but as far as I can tell, they are all commercial and/or expensive.
