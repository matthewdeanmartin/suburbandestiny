title: Idea for using WordPress as a RSVP engine
---


1\) Event hosts post event  

2\) Site owner reviews event, fixes punctuation, spelling etc.  

3\) People RSVP in the comments.  

4\) Special tag shows event date, all posts on blog sorted by event date, so it becomes it’s own calendar.


What could be easier?
