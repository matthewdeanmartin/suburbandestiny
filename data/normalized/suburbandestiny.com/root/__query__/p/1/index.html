The least read post

Posted on January 15, 2006 by matt

The first post is the least read post, so writing it is almost an exercise in futility. This site has passed through many incarnations. At first, I wanted it to be a replacement for meetup.com. However, that didn’t work, because I didn’t have TV ads, radio ads, or a paid staff to promote the website. The second incarnation was a blog used as an social event posting site. That is still available at https://suburbandestiny.com/events/

I also have a blog on this site, which originally was a personal blog and got many hits because I used to be an avid commenter on other sites. Now that I don’t comment, I don’t get any comments in return. And I started mixing technical notes with notes about my personal life. This front page blog is only going to be about my personal life.
